Question Group 1

Q1: What are the principles of analysis and design?

Analysis is an investigation of the problem and requirements, and Design is a conceptual solution that fulfills the requirements.

Q2: What are website architectural requirements?

Website architecture is the planning and design of the technical, functional and visual components of a website and used in creating a logical layout of a website in line with the user and/or business requirements. Website architectural Requirements are: * Technical constraints; server, storage, memory, and communication interface. * Functional aspects; the types of services, processes the website will provide. * Visual appearance; the user interface, colours, buttons, and other visual design elements. * Security parameters; how the website will ensure access control and transactions.

It takes advantage of a very simple top-down approach to guide from main page (home page) to subpage (child page) with specific content.

Q4: What are user-interface design requirements and production processes?

User Interface design focuses on anticipating what users might need to do and ensuring that the interface has elements that are easy to access, understand, and use to facilitate those actions. The process of the UI design concepts includes interaction design, visual design, and information architecture. UI design requirements are * Input controls: Buttons, text fields, checkboxes, radio buttons, dropdown lists, list boxes, toggles, data field etc * Navigation components: icons, breadcrumb, slider, search field, pagination, tags. * Informational components: tooltips, icons, progress bar, notifications, message boxes, modal windows.
