Question Group 2

Q1: What are programming controls and design structures?

Programming controls is a programming paradigm aimed at improving the clarity, quality, and development time of a computer program by making extensive use of the structured control flow constructs of selection (if/then/else) and repetition (while and for) block structures, and subroutines. Design structures is called structured design and can be described with four logical structures. When using the structures in a disciplined manner, the algorithm described in the pseudo-code is called a structured algorithm.

Q2: What are website testing procedures?

The website testing procedures consists of multiple steps that ensure that an application is fully functional and runs smoothly and securely. It is an essential part of web development and ensure that an application is running properly before its release.
There are 6 steps as follows:
Step 1: Functional Testing
Step 2: Usability Testing
Step 3: Interface Testing
Step 4: Compatibility Testing
Step 5: Performance Testing
Step 6: Security Testing

Q3: What are website debugging methods?

Debugging is a methodical process of finding and reducing the number of bugs, or defects, in a computer program. A lot of web development tools are integrated into browsers. These tools are not just for web development, but also serve a lot of other functions, of which debugging is one of the most prominent. These tools help inspecting and analysing possible issues with the CSS, HTML, and Java Script used in the code.
